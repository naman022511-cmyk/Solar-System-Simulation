Solar System Simulation

This is a 2D simulation of the solar system made with Java Swing. Uses circular motion and spaced distancing.
Mostly a passion project I made to help me learn about physics and trigonometry.

Features

1. Orbital Motion using trigonotmetry

2. Separate classes for all planets of the solar system and the sun.

3. Circular Orbits

4. Scaled distances and sizes

5. Languages and Libraries Used

6. Java

7. Java Swing

8. Graphics2D


Improvements

1. Elliptical Orbits

2. Using images for the planets

How to Run

1. Download the Repository

2. Open the project in a Java IDE
   
3. Run the main class
